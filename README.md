# Meal Planner

A Python Tkinter based desktop application that helps users plan meals, manage recipes, and organize daily meal plans through a simple graphical interface.

## Features
- GUI built with Tkinter
- Create and manage meal plans
- Store meals and recipes using JSON files
- Simple and user-friendly desktop interface
- Lightweight and runs locally (no internet required)

## Technologies Used
- Python 3
- Tkinter (GUI)
- JSON for data storage
